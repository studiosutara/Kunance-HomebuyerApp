What's in it?

It is a native iOS application written completely in Objective-C. Uses

  • Shinobi charts for graphing
  • Cocoapods for various libraries

Cocoapods used

  • Facebook iOS SDK
  • HTAutocompleteTextField
  • MBProgressHUD
  • PKRevealController
  • SPGooglePlacesAutoComplete
  • TSCurrencyTextField
